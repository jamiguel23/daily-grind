This project is a dynamic webpage that changes what is rendered based on the time of day. The Date function with the getDay() method was used as an argument for the switch statement. The cases were numbered and a coffee variable object was assigned key value pairs that determined what was rendered on the page. A function was created to return html that would be inserted using the document.getElementbyId method helping with the dynamic rendering. the function was also used for the querystring so that when a selected link to the special of the day of the week is clicked, the page renders that special on days it does not fall on.
